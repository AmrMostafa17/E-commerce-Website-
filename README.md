# 🛒 E-Commerce Web Application (Amazon Clone)

## 📌 Overview

This project is a dynamic e-commerce web application inspired by Amazon. It provides a complete online shopping experience where users can browse products, search, filter by categories, add items to a cart, and place orders بسهولة.

---

## 🚀 Features

* 🔍 Search for products
* 🗂️ Filter by categories
* 📄 Product details page
* 🛒 Add to cart (LocalStorage)
* ➕➖ Update quantity & remove items
* 💳 Checkout system
* 🧾 Store orders in database (orders & order_items)
* 🌙 Dark / Light mode toggle
* 📱 Responsive design

---

## 🛠️ Tech Stack

* **Backend:** PHP
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
* **UI Framework:** Bootstrap

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/AmrMostafa17/E-commerce-Website-/tree/main
```

2. Move project to:

```
htdocs (XAMPP)
```

3. Import database using phpMyAdmin

4. Configure database connection in:

```
inc/config.php
```

5. Run the project:

```
http://localhost/E-commerce_Website_Final_Project
```

---

## 🧾 Database Tables

* products
* categories
* orders
* order_items

---

## 🛍️ How It Works

1. User browses products
2. Adds items to cart
3. Manages cart (update / remove)
4. Proceeds to checkout
5. Enters details
6. Order is stored in database
